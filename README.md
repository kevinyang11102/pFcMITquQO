# 图书管理系统 python488

一款基于 Python + Flask + MySQL 开发的图书馆管理系统，旨在提供高效的图书管理解决方案。

## 技术栈

- 后端语言：Python
- 框架：Flask
- 数据库：MySQL, SQLite
- 开发工具：PyCharm
- 前端UI框架：Layui
- 数据库连接工具：Navicat

## 功能模块

### 系统角色

- 管理端
- 用户端

#### 用户端

- 用户登录与注册

#### 管理端

- 图书信息管理（增、改、删、查）
- 可视化数据展示
- 用户图书采购需求管理

## 系统运行环境

- 操作系统：Windows

## 部署步骤

1. 导入项目到 PyCharm
2. 安装所需依赖包
3. 配置 MySQL 或 SQLite 数据库
4. 运行系统

## 目录结构

```
(root)
│
├── app/               # Flask 应用目录
│   ├── models/        # 数据模型目录
│   ├── views/         # 视图处理目录
│   ├── templates/     # HTML 模板目录
│   ├── static/        # 静态资源目录
│   └── ...
│
├── venv/              # 虚拟环境目录
├── requirements.txt   # 依赖包列表
└── ...
```
